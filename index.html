HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Cascading Style Sheets (CSS) is a programming language used by developers to define how webpage content is presented to users. Defines the colors, aesthetics, visual layout, and organization. Defines style rules for specific elements of an HTML page and format them using a wide range of style options.
  • INLINE CSS: to style an element directly in the HTML file using a style attribute with a CSS rule to style a particular page element only.
  • INTERNAL CSS: embed an entire stylesheet directly in the HTML file using a style element in the head element.
  • EXTERNAL CSS style sheet: allows developers to keep all their CSS rules in a separate file, which makes design changes easier.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • pwd: displays the current working directory
  • ls: displays the list of files and folders in the current directory
  • cd: navigates between directories in the path
  • cd ..: navigates backwards, one level at a time, through the path of your directories
  • mkdir: creates a new directory
  • touch: creates a file inside a directory
  • start: opens a file

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script (top to bottom).
  • To declare a variable, we use the varkeyword. After the var keyword, we give the variable a unique name. Finally, we use an assignment operator = to assign the specific data that we want stored in our variable.
  • We can use the console.log method to output a message to the web console by adding an argument.
  • Ctrl+shift+I to open Developer Tools
  • A value is considered falsy if it is one of the following values:0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false.
  • All other values, including a Boolean with the value of true, are considered truthy.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
